```text
main.go: 程序入口
go.mod: 第三方包存放位置
setting.yaml: 配置文件
api: 接口目录
config: 存放记录配置的结构体的目录
core: 初始化操作
docs: swager生成的api文档目录
flag: 命令行相关
global: 全局变量的包
middleware:中间件
models: 表结构
routers: gin路由的目录
service: 项目与服务有关的目录
testdata: 测试文件的目录
utils: 常用的一些工具
.gitignore: git忽略文件

```
